基础篇

1.基础知识

1.1 历史等

(1) C语言的特点

  • 贴近硬件
  • 面向过程：C语言以过程为中心的变成范式
  • 面向对象：以对象为核心的编程范式

(2) C++特点

  • C的基础
    • 贴近底层，运行速度快
    • 编译型语言
    • 静态类型语言
    • 结构化教学语言
  • 增强半C语言
    • 面向对象编程
    • 面向泛型编程
    • 功能强大，相对较难掌握

(3) 编译和解释型语言

  • 编译：由编译器把整个源代码翻译成机器码，最终生成二进制文件，一次性提交给计算机 - 代表C、C++
  • 解释：由解释器将代码逐行解释成机器码，并交给计算机执行 - Python、JavaScrip

(4) C++代码的运行

  • 源代码(.cpp) =编译=> 目标代码(.obj) =(库文件->)链接=> 可执行代码(.exe)

(5) C++标准

(6) C++编译器

  • Visual Studio

1.2 基本常识

(1) 常识性

  • “#”：预处理，程序运行之前预先要做的事情
  • 别的文件的函数名直接放在要调用的上边，因为编译器能够找到编译后的函数，只要给返回值和参数类型
  • “std”命名空间：存放cout、cin、string、vector等
  • “::”：告诉编译器，要用某个特定作用域里边的名字

1.3拓展 “::” 的作用和使用

1.3.1 作用域解析运算符 - 四大核心用途

(1) 访问命名空间内的成员
  • 告诉编译器，名字在std命名空间内
  • 可能有许多库都有string的结构
(2) 访问类/结构体/枚举的成员
(3) 访问基类成员(父类成员) - 在多继承或子类重写函数时特别重要：
(4) 访问全局作用域 - 前面单独加::
  • 单独的 :: 代表 全局命名空间（global namespace）。
  • 用途：避免局部变量、函数名遮蔽全局的同名函数。
(5) ::的意义
  • C++具有复杂性
    • 多重命名空间
    • 多个类拥有同名函数或变量
    • 多重继承
    • 局域名与全局名可能冲突
    • 外部库的名字可能相同
  • 所以指出具体这个名字是从哪来的

2.变量常量等

2.1 变量 - variable

2.1.1 基本知识

(1) 变量四要素
  • 类型：int float doouble char bool class等
  • 名字：变量名需以字幕或下划线开头(不能使用双下划线和单下划线加大写字幕)
  • 存储周期：自动(auto)、静态(static)、动态(new)
  • 作用域：全局、局部、类作用域、命名作用域
(2) 类型分类体系
  • 按基本类型分
    • 基本类型：int float fouble char bool
    • 符合类型：指针、引用、数组、结构体、类、枚举
  • 按存储周期分
    • 自动变量(auto)：进入作用域创建，离开作用域销毁
    • 静态变量(static)：程序开始创建，程序结束销毁(全局生命周期)
    • 动态变量(new)：
  • 按存储位置
    • 栈变量(stack) 局部变量
    • 堆变量(heap) new出来
    • 全局/静态区 全局变量、static变量
(3) 变量初始化
  • 直接初始化 - 允许隐式类型转换，会导致意外行为
    • int a(10);
  • 括号初始化 (统一初始化) - 会阻止隐式类型转换，防止精度丢失
    • int b={20};double c{3.14};
  • 复制初始化
    • int d=30;
  • 默认初始化
    • int x;

1.4 常量 - 值不能变的存储单元

(1) 分类

  • 字面量常量
  • const常量 - 编译期只读变量
  • constexpr - 编译期常量
    对比项目 const constexpr
    必须在编译期确定？ ❌ 不必须 ✔ 必须
    可用于数组大小？ 不一定 ✔ 必须可用
    适合场景 运行期也可能用 编译期优化
  • 宏从常量 - 预处理常量
    • #define - 宏定义

(2)

  • const限定符 -
    • const int Zero = 0;

1.5 数据类型 四类

1.5.1 基本类型 Built-in Types

  • bool >>

  • char >> 1字节
  • wchar_t >>
  • char16_t/32_t
  • int >> 4字节
  • long >> 4或8字节
  • long long >> 8字节
  • float >> 4字节
  • double >> 8字节
  • long double >>
(1) 浮点类型
  • double长度不小于float
  • 提供科学计数法(E表示法)：5.98E24 == 598×10^24

(2) 复合类型(Compound Types)

  • 指针 pointer
  • 引用 reference
  • 数组 array
  • 函数指针
  • 类 class/结构体 struct

(3) 用户自定义类型(User-defined Types)

  • struct：纯数据，不带函数
    • struct Point{int x,y;};
  • 枚举 enum：enum Struct{ST_INIT,ST_NORMAL};
  • typedef / using(类型别名)
    • typedef unsigned char u8;
    • using u16 = uint16_t;

(4) 类型修饰符(Type Modifiers)

  • signed 有符号(默认)
  • unsigned 无符号
  • const 只读，放 Flash 中
  • volatile 不优化(用于寄存器、RTOS 共享变量)
  • static 静态存储/内部链接
  • extern 声明外部变量
  • mutable const 对象中可修改
  • auto 自动类型推导
  • register 基本已过时
